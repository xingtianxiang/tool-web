# 加工件采购分发管理 — 使用说明

一个本地、离线的桌面小工具,帮你管住「哪一版图纸发给了哪家厂商、发没发」。数据全在你电脑上,不上网、不登录。

## 怎么运行

双击 `dist\加工件采购分发管理.exe` 即可打开,不用安装、不用装环境。

> 如果是从别的电脑拷过来第一次运行,Windows 可能弹「已保护你的电脑」——点「更多信息 → 仍要运行」即可(这是因为没买代码签名证书,自用没必要)。

首次打开会自动在 **文档\加工件管理** 下建一个数据文件夹,所有零件、图纸、记录都存这里,可随时整体复制备份。

## 日常流程

1. **零件**页:点「新增零件」,填图号/名称和需求(材料、数量、公差、交期、表面处理)。
   - 一个零件可以有**多个图纸文件**(2D图、3D图、几个子零件图)。把文件**拖到零件卡片上**(可一次拖多个),或点「添加文件」——是**追加**,不会覆盖。
   - 每个文件可点「标签」起名(如 2D图 / 3D图 / 子零件A),点「替换」改版(旧文件自动留底到「历史文件」),点「删除」移除。
2. **厂商**页:点「新增厂商」,把要对接的厂商加进来。
3. **状态矩阵**页:行是零件、列是厂商。**点格子**就把某零件指派给某厂商。
4. 在某厂商列顶点「**打包**」→ 核对清单 → 「生成压缩包」。
   - 工具会把该厂商名下所有零件的**最新图纸** + 一份自动生成的 **需求单.pdf** 打成一个 zip。
   - 生成后点「打开所在文件夹」,把这个 zip **拖进微信**发给厂商即可。

## 关键:图纸改了怎么办

在**零件**页找到那个文件,点「替换」上传新图(旧图自动留底);或者「添加文件 / 删除文件」。只要零件的图纸有任何增/换/删,它的**修订号就 +1**,这一刻**状态矩阵里所有还拿着旧修订的厂商格会变成黄色 ⚠️ 需重发**(显示 v1→v2)。看到黄色就重新打包发一次,绿色 ✓ 就是已经发了最新。再也不用靠脑子记。

## 状态颜色

- ✓ 绿色:已发最新版
- ⚠️ 黄色:图纸更新了、该厂商还拿着旧版,要重发
- ○ 灰色:已指派,还没发过
- — :没指派给这家厂商(点一下即可指派)

## 备份

**设置**页 →「导出备份」,会把所有数据打成一个 zip。或者直接到 **文档\加工件管理** 把整个文件夹复制走。建议偶尔备份一次。
